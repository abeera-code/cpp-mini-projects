# C++ Mini Projects

## Project 1: Employee Management System

### Features:
- Add Employee
- Display Employees
- Calculate Salary with Bonus
- Copy Employee Record
- Show Total Employees

### Concepts Used:
- Classes & Objects
- Constructors (Default, Parameterized, Copy)
- Static Variables
- Arrays of Objects
- Menu Driven Programming

### Description:
This project is a simple Employee Management System built in C++. It allows users to manage employee data and calculate salary based on attendance.
